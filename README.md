# 🧮 Math Game

Jogo de quiz matemático no console feito em C#, onde o jogador responde perguntas e acumula pontos.

## 🎮 Funcionalidades

- Quatro operações: Adição, Subtração, Multiplicação e Divisão
- Modo Jogo Aleatório com operações misturadas
- Níveis de dificuldade (Fácil, Médio, Difícil)
- Cronômetro por partida
- Histórico de partidas salvo durante a execução
